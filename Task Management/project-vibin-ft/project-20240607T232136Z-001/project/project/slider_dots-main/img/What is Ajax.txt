What is Ajax?
Ajax is an acronym for Asynchronous Javascript and XML. 
It is used to communicate with the server without refreshing the web page and
 thus increasing the user experience and better performance.


Ajax allows web pages to be updated asynchronously by exchanging small amounts of
 data with the server behind the scenes. 

BENEFITS OF USING AJAX:

XMLHttpRequest is a request type widely used for sending a request to Ajax pages. 
You can also call it with a different name: Asynchronous HTTP request.
 It plays a vital role in the implementation of Ajax techniques for web development.
